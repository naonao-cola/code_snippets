# 护萌宝
---
### 战舰少女R 辅助软件

本软件为本人初学编程时的练手做, 因此结构混乱, 代码冗杂, 耦合度极高, 命名混乱, 不再具有可维护性与维护价值, 本人已经放弃维护.

现在已存在魔改版, 请自行搜索

**适用版本: 理论上`4.X`通用**

### 目前功能
- 自动远征
- 战役演习泡澡
- 定时操作
- 自适应版本号

### 目前有问题功能
- 轮换功能会出现不存在错误
- 出征可能出现未知错误
- cpu占用偏高
